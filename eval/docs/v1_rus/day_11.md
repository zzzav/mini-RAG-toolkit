# День 11 — ExtractLLM (ответ из контекста без внешнего LLM)

## Что сделали
- Добавили режим `--llm extract`:
  - если контекст пустой или совпадений нет → “В контексте нет информации.”
  - иначе: режем контекст на предложения, скорим по токенам вопроса, берём top_n
  - сохраняем порядок контекста (pos) при равных score
  - дедуп предложений через list + seen
- Улучшили UX:
  - inline-citations в text режиме (по желанию)

## Тесты
- `test_extract_empty_context`
- `test_extract_picks_relevant_sentences`
- `test_extract_no_overlap`
- E2E с tmp_path для документов.

## Результат
Появился “RAG-правильный” ответ без LLM API: он извлекает информацию из контекста, не фантазирует.
