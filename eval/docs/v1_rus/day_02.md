# День 2 — CLI: text_tool (нормализация и статистика текста)

## Что сделали
- Реализовали CLI-утилиту `src/text_tool.py`.
- Поддержали источники ввода:
  - `--text` (строка)
  - `--in` (файл)
- Добавили обработку текста:
  - `normalize_text` (приведение к единому виду)
  - подсчёт статистики
  - `--upper/--lower` (приоритет у `upper`)
  - `--stats-only`
  - `non_empty_lines`
- В чтении файла использовали `errors="replace"`.
- Обработку ошибок CLI сделали через `SystemExit(2) from None`.

## Тесты
- Написали тесты `tests/test_stats.py` на ключевые ветки.

## Результат
Появился первый “настоящий” CLI с понятными флагами, устойчивыми ошибками и тестами.
