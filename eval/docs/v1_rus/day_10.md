# День 10 — Citations + единый результат RAGResult

## Что сделали
- Начали реально использовать `RAGResult`:
  - один объект результата: query, chunks, context, prompt, answer, citations
- Добавили `collect_citations()`:
  - уникальные (source, idx)
  - стабильный порядок (по первому появлению)
- Обновили CLI:
  - `--format text|json`
  - вывод `CITATIONS` отдельным блоком
  - (опционально) inline citations для копипаста

## Тесты
- Проверки уникальности и порядка citations.
- E2E сборка результата.

## Результат
RAG-пайплайн стал структурированным: результат можно сохранять, сравнивать, анализировать.
