GDPR & DATA PROCESSING

1) GDPR data processing agreement
We offer a Data Processing Agreement (DPA) for customers acting as controllers under GDPR.
The DPA defines roles, processing instructions, security measures, and subprocessors.

2) Subprocessors
A list of subprocessors is available upon request and is updated when changes occur.
We notify customers in advance of material changes where required.

3) Data subject requests
We support requests for access, rectification, erasure, restriction, and portability where applicable.
